# Content Studio

A full-stack YouTube content creation dashboard. Manage your entire pipeline from research to publishing - powered by AI.

**Features**: Video pipeline, thumbnails, multi-platform publishing, repurposing, shorts, carousels, analytics, peer tracking, AI news digest.

## Quick Start

See [SETUP.md](SETUP.md) for manual setup, or use Claude Code:

```
git clone https://github.com/bosar-youtube/content-studio.git
cd content-studio
```

Then in Claude Code:
```
Read the CLAUDE.md and guide me through the complete setup.
```

## Tech Stack

Next.js 16 + React 19 + Supabase + Anthropic Claude + Google Gemini + Tailwind CSS + shadcn
